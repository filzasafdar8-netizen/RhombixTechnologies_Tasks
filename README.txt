Project Name: Basic Network Sniffer

Description:
This project is a basic network sniffer developed using Python.
It captures live network packets and displays IP addresses and protocols.

Tools Used:
Python (Anaconda), Scapy

How to Run:
1. Open Anaconda Prompt as Administrator
2. Install scapy: pip install scapy
3. Navigate to project folder
4. Run: python sniffer.py
